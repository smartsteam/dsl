마이크로비트(micro:bit)를 연결하려면 아래 버튼을 누르세요.
※ MakeCode에서 'Bluetooth UART 서비스'가 켜진 hex가 필요해요
🛠️
마이크로비트 BLE hex 만들기
마이크로비트와 연결됐지만 BLE 센서 서비스가 활성화되지 않았어요. MakeCode에서 hex를 새로 만들어 플래시해 주세요.

📝 단계별 따라하기 (5분 소요)

  1. 크롬에서 makecode.microbit.org 접속 → 새 프로젝트 클릭
  2. 오른쪽 위 ⚙️ 톱니바퀴 → 확장 → "bluetooth" 검색 → 클릭하여 추가
    ※ 빨간 경고가 사라지고 메뉴에 'Bluetooth' 카테고리가 추가돼요
  3. 오른쪽 위 ⚙️ → 프로젝트 설정 → "No Pairing Required"를 선택 → 저장
    ※ 페어링 없이 빠르게 연결할 수 있게 해줘요
  4. 왼쪽 위 {} JavaScript 버튼을 눌러 텍스트 모드로 전환
  5. 편집기의 모든 코드를 지우고, 아래 코드를 그대로 붙여넣기:
bluetooth.startUartService()
basic.forever(function () {
    bluetooth.uartWriteLine("" + (input.temperature()))
    basic.pause(1000)
})
  1. 왼쪽 아래 다운로드 버튼을 눌러 .hex 파일 받기
  2. 마이크로비트를 USB로 PC에 연결 → hex 파일을 MICROBIT 드라이브에 끌어다 놓기
    ※ 마이크로비트 뒷면 LED가 잠깐 깜빡이면 플래시 완료
  3. 이 페이지로 돌아와서 🔴 해제 → 🔗 장치 연결 → 같은 마이크로비트 다시 선택
  4. ▶ 수신 시작 클릭 → 그래프에 온도 데이터가 흐르면 성공! 🎉

🔧 다른 센서/여러 값 보내기

코드의 input.temperature() 자리를 바꿔주세요:
  • input.lightLevel() — 💡 밝기 센서 (0~255)
  • input.compassHeading() — 🧭 방위각
  • input.acceleration(Dimension.X) — 📐 가속도 X축
  • input.soundLevel() — 🔊 소리 (v2 마이크로비트만)
💡 Tip — 채널 라벨을 의미 있게 표시하기
시작 시 한 번만 @schema 라인을 보내면 그래프 라벨이 자동으로 의미 있게 표시돼요:
bluetooth.uartWriteLine("@schema,1,mb:S22:CH1,mb:S25:CH1")
S22=온도, S25=빛 등 — 각 슬롯마다 의미 있는 이름을 갖게 돼요.
여러 값 동시에 보내기 (자동으로 다채널 그래프 생성):
bluetooth.uartWriteLine(input.temperature() + "," + input.lightLevel())
